команда - описание, возвращаемое значение

set ON - включить макет
set OFF - выключить макет
get S # - запрос температуры на термометре #, градусы С
get F # - запрос скорости потока на датчике #, л/с
set P # M - установить мощность M на насосе #, у.е. 0..1
set H # M - установить мощность M на кипятильнике #, у.е. 0..1
set L R G B - установить цвет светодиодов RGB

Дополнения:

-1 - пустышка
пар и псевдо-турбины включаются автоматически
в случае отсутствия элемента отправлять пустышку (?)
